Scratch Challenges

Scratch Wiki - This wiki provides explanations about various concepts in the Scratch programming language.

#1 - Hello, World!

Create a Scratch project with a cat sprite that meows once and displays the message "Hello, World!".


Did you know? "Hello, World!" is one of the most basic programs to write in any programming language.


#2 - Personalized Greeting

Create a Scratch project with a cat sprite that asks the user for their name and have the cat then say "Hello, [name here]!" after the user has submitted their name in the prompt.


This challenge introduces you to the concept of variables and assigning variables values. Variables are storage locations for data in a program. You can think of it as an envelope (like pictured below) where the name of the variable is the name on the envelope, and the contents inside are the data stored in the envelope. Consider creating the variable name and assigning it the value the user gives in the prompt.





#3 - Conditional Greeting

Modify the Personalized Greeting so that the cat will only greet you if your name is entered; otherwise, the cat should refuse to greet the user.


This challenge introduces you to the concept of if-else conditions. The if statement checks first whether one condition is true then does something, else if this condition is not true, we do something else.





#4 - Bouncing Basketball

Select the basketball sprite to use for this program and delete the cat sprite. Have the basketball move forever and if the ball is at the edge, then have it bounce off the edge.


This challenge introduces the concept of an infinite loop. Consider this as an infinite loop:





#5 - Count to Ten

Have a sprite of your choice count to ten. Use a variable for count and increment this variable using a while loop (repeat until block in Sratch).


This challenge introduces the concept of a while loop. A while loop repeats some group of actions while a certain condition is true.





#6 - Glide Around Screen

Make the cat glide around the edge of the screen.


#7 - Walking Cat

Add your own backdrop to the stage and have the cat walk forever back and forth at the bottom of the stage.


#8 - Random Mouse

Add a mouse sprite and have it move forever in sometimes random directions, bouncing on the edges when it touches the edges.


#9 - Cat and Mouse

Modify the Random Mouse program by adding a cat. If the mouse touches the cat, broadcast the message to the cat sprite so that the cat shows the message, "Yummy!", and the mouse disappears and reappears at a random location on the stage. Hint: you'll have to have add another thread for the mouse which begins when the green flag is clicked.


#10 - Cat and Mouse - Interactive

Modify the Cat and Mouse program by making the cat sprite interactive. Another thread should be added to the cat, so that it senses whether the up, down, left, or right arrow keys are pressed. If the up key up arrow key is pressed, the cat should move some steps forward. If the down arrow key is pressed, the cat should move some steps backward. If the left or right aroow keys are pressed, the cat should rotate some.


Your own project

Create your own Scratch project and add it to our Coding 101 Guild studio. In your project, try to include the following:

  • At least two sprites, one which is not a cat.
  • At least three scripts total.
  • At least one condition, one loop, and one variable.
