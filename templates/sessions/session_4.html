Create your own website

  1. Make an account for yourself at Codecademy if you do not have one already.

  2. Work through the tutorial About You - Make a website all about you. At the end of the tutorial you should have an "index.html" file that renders something like this:




  3. Create an account for yourself at PythonAnywhere and confirm your account through your email address. PythonAnywhere allows you to "host, run, and code Python in the cloud" which basically means we can create and deploy a Python web application right in the browser.

  4. After creating an account, navigate to the "Web" tab in the navigation bar and click "Add a new web app".




  5. The basic plan for PythonAnywhere doesn't support custom domain names, i.e. the web address of your site, so click "Next" on this step.




  6. The next step is to select a Python Web framework. A web framework can be thought of as a model for how a website is developed as well as a tool set for web development. We will be using the Flask framework (actually, we use Flask for many of our Python microservices at Stylight!) because it is lightweight and easy to learn. Select "Flask" from the list of frameworks and click "Next".




  7. Choose "Python 3.5" from the list of Python versions and click "Next".




  8. On the "Quickstart new Flask project" page, you can change the name of the folder path where you app exists; however, we'll just use the default path, so you do not need to change anything here. Click "Next".




  9. You should now see a page like the one below:




  10. Navigate to the "Files" tab on the navigation bar. On the left side of this page, you can see your directories (folders) and on the right, the files in the opened directory. You can see the directory path on the top left under "Consoles." In this example. the opened directory is "/home/coding101".




  11. Under the "Directories" view, open the directory called "mysite/". You should then be able to see the Python called "flask_app.py" under the "Files" view.




  12. Open up the "flask_app.py" Python file. This opens a simple Python IDE (Integrated Development Environment) which is simply a text editor that helps for Python development. In the file, there is already code for "Hello World". On line 8, change "@app.route('/')" to be "@app.route('/hello')" as shown below, and click "Save" on the upper right corner.




  13. Navigate back to the "Dashboard" (located on top right corner of page). And then navigate to the "Web" tab. Under the "Web" view, click on the yellow button, "Run until 3 months from today." This begins your website for a period of three months. Then click the green button, "Reload ..." to deploy your website.




  14. In your browser address bar, navigate to your website followed by the path "hello/" which should be username.pythonanywhere.com/hello, e.g. coding101.pythonanywhere.com/hello. You should then see, "Hello from Flask!" on the webpage.




  15. Congratulations! You now have your own website ☺. Let's now add your Codecademy masterpiece to your website. Navigate back to your files and under the "Directory" view and go into the directory "mysite/", then type "templates" in the input "Enter new directory name" and click "New directory".




  16. You can see from the file path, that you are now in the new directory you created called "templates". Under the "Files" view, type "index.html" in the "Enter new file name" input and them click "New file."




  17. The text editor should now be opened for the file "index.html". You should now copy & paste the contents from your Codecademy "index.html" file into the editor like below and click "Save":




  18. Navigate to "Files" and then open up to the directory "mysite/" and then open the Python file "flask_app.py". Modify the Python file so that it is identical to the file below. Be sure to add the "from flask import render_template" at line 5 and lines 13-15. Basically, we are adding a Python function that will render our HTML template, in this case "index.html". Click "Save" when done.




  19. Navigate back to "Web", click the green "Reload username.pythonanywhere.com" then open your webpage username.pythonanywhere.com, e.g. coding101.pythonanywhere.com. You should see the same HTML page you created at Codecademy. If you have a 404 Not Found or 500 Internal Server error, ask for help.




  20. Voila! You can now play around and make changes to your index.html, reload, and then instantly see the changes on your new website. Have fun!
